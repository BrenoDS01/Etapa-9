# Projeto PI - Raquel Fit

Este é o projeto **Raquel Fit**, uma aplicação desenvolvida com **Java Spring Boot** para gerenciar alunos, pagamentos e presenças.

## Funcionalidades

- Cadastro de Alunos
- Controle de Pagamentos
- Controle de Presenças
- Dashboard com resumo de informações
